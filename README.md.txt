UZDUOTIS

Padauginyti skaicius is pateikto duomenu masyvo.
Is padaugintos sumos atimti duomenu masyvo ilgi (array.length).
Skaiciavimas turi buti atliktas atskirose funkcijose "daugyba" ir "atimtis".
Reikalavimas, jog kiekvienoje funkcijoje buti atliekamos visos reikiamos validacijos.
Naudojame import-export JS funkcionaluma.

failai:
- data.js
- atimtis.js (pagrindine kvieciama funkcija)
- daugyba.js   (skaiciuoja masyvo skaiciu daugyba)
- main.js

demo duomenys:
[5, 1, 4, 6, 2, 7, 14]

tikimasi rezultato:
"Is pateiktu 7 skaiciu, galutinis rezultatas po daugybos ir atimties gaunasi: 23,513."
